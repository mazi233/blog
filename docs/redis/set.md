# Set类型操作 (不重复的数据)
## sadd (添加)
> sadd set value
```
sadd set1 11
```

## srem (删除)
> srem set value
```
srem set1 11
```

## sismember (查看某条数据是否在set中)
> sismember set value
```
sismember set1 11
```

## scard (长度)
> scard list
```
scard list1
```
